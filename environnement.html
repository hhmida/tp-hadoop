    Préparation de l'environnement¶

    Installation de Virtual Box¶

    Télécharger Oracle Virtual Box (ou VMware) selon votre système (Windows, Linux ou Mac)

    Démarrer l'installation de Virtual Box

    Suivre les étapes ci-dessous

    Créer la machine virtuelle¶

    Télécharger l'image de la machine virtuelle

    En utilisant VirtualBox et à partir du menu Fichier -> Importer un appareil virtuel, sélectionner le fichier BDTools.ova fourni et choisissez l'emplacement de destination.

    Une fois l'importation finie, vérifier les paramètres de la machine virtuelle pour les ajuster à la configuration de votre machine. Il est recommandé d'utiliser 8G de RAM en gardant au moins 2G pour la machine hôte.

    Démarrer et tester la machine virtuelle¶

    Démarrer la machine virtuelle et se connecter avec l'utilisateur uti. Le mot de passe =uti=.

    Lancer le navigateur web, la page http://127.0.0.1 sera chargée automatiquement où l'état des services est affiché :

    Par défaut, les services actifs sont HDFS, YARN et HBase. La commande suivante permet de sélectionner les services et activer/désactiver pour optimiser la gestion de la mémoire :

    sudo SelectService hadoop | spark | cassandra | elasticsearch
